## 1. PRICING QUICK REFERENCE

**Value Benchmark: €30,000 conservative value in 6 months**
(Half a client + time savings = almost guaranteed)

```
ANCHOR HIGH:     €1,000 advance + €1,200/mo + 15%  →  28% of value  →  260% ROI
TARGET:          €750 advance + €1,000/mo + 10%    →  23% of value  →  335% ROI
FLOOR:           €500 advance + €800/mo + 5%       →  18% of value  →  450% ROI

Tool budget:     €150 (one-time)
Running costs:   €50/month
```

**Say this:** "You're paying 20-25% to capture value you're leaving on the table."

<a name="objections"></a>
## 2. OBJECTIONS → RESPONSES

**"Why an advance?"**
→ "Pre-payment for 60 hours of dev. You're getting a discount rate."

**"Too expensive"**
→ "One client = €35K. System costs €15K/year. 230% ROI minimum."

**"I need to think about it"**
→ "What's the one thing holding you back?"

**"Can we start smaller?"**
→ "The advance funds the MVP build. After that, just the retainer."

**"Your tool is like mine"**
→ "Yours finds emails. Mine sends, personalizes, follows up, and tracks. Step 1 vs steps 1-10."

**"What if it doesn't work?"**
→ "Refund clause in the contract. If I don't deliver MVP, you get money back."

**"Can I pay later?"**
→ "Advance due in 3 days of signing. First retainer due month 2."

---

### ⚠️ NO COMPANY OBJECTIONS

**"Company isn't registered yet"**
→ "No problem — we can finalize the contract once it's ready. For now, let's agree on the terms. You could pay the advance personally, or we wait for registration."

**"Let's start without the advance"**
→ "I need to cover tools and dev time upfront. Once payment clears, I start within 48 hours. The refund clause protects you."

**"Can we just do a trial first?"**
→ "The advance IS the trial — 6 weeks to MVP. If I don't deliver, you get your money back."

**"I'll pay you when the company is set up"**
→ "When do you expect that to happen? [If >2 weeks] That works — we can sign then. [If vague] Would you consider paying personally now and getting reimbursed by the company later?"

<a name="math"></a>
## 4. ROI MATH

**Use his numbers against him:**

```
His deal size:          €35,000 (average)
System cost (year 1):   €12,000 (advance + 12 months)
Break-even:             0.35 clients

→ If system helps close 1 extra client = 290% ROI
→ If 2 extra clients = 580% ROI
```

**Time savings:**

```
His time on outreach:   10 hrs/week
His hourly value:       €50-100/hr
Weekly savings:         €500-1,000
Monthly savings:        €2,000-4,000

→ Time savings alone > your retainer
```

<a name="contract"></a>
## 5. CONTRACT HIGHLIGHTS

**If he asks about the contract:**

| Topic | Answer |
|-------|--------|
| Length | 3 months initial, then month-to-month |
| Termination | 14 days notice either side |
| MVP deadline | 6 weeks from payment |
| Refund | Pro-rata if YOU fail to deliver |
| No refund if | HE changes scope, goes silent, or terminates |
| IP | He owns system after full payment |
| Jurisdiction | Paris or Lisbon (his choice) |

**Key protection for him:**
- Refund if you disappear or don't deliver

**Key protection for you:**
- No refund if he ghosts, changes scope, or terminates early

<a name="next"></a>
## 6. IF HE SAYS YES

### If company IS registered:
1. ✅ Walk through contract highlights together
2. ✅ Fill in: amounts, dates, jurisdiction
3. ✅ Sign on call (electronic is fine)
4. ✅ Send invoice for advance immediately
5. ✅ Confirm MVP deadline (6 weeks from payment)
6. ✅ Schedule first weekly check-in

### If company NOT registered yet:
1. ✅ Agree on terms (amounts, scope, timeline)
2. ✅ Ask: "When will the company be registered?"
3. ✅ Option A: He pays personally now → you start
4. ✅ Option B: Wait for registration → sign + pay then
5. ✅ Get commitment in writing (even informal email)
6. ✅ **DO NOT START WORK UNTIL PAYMENT CLEARS**

## EMERGENCY: IF GOING BADLY

**He's cold / skeptical:**
→ Go back to questions. "What's the biggest bottleneck for growth right now?"

**He's comparing to agencies:**
→ "They don't build you a system you own. I do."

**He's stuck on price:**
→ "Let's do the math. What's one client worth? [€35K] So the system pays for itself with one-third of a deal."

**He's not engaging:**
→ "I want to make sure this is right for you. What would make this a no-brainer?"

## YOUR MINDSET

- You're not begging. You're offering value.
- He NEEDS this. You're helping him see it.
- If he says no, you walk. There are other clients.
- Your floor is €500/€800/5%. Below that = no deal.
- **No company is not a dealbreaker — but no advance IS.**
- He's making €150K+ ARR. He has money. The question is commitment.

---

## ⚠️ RED FLAGS — WALK AWAY IF:

- He wants you to start working before paying
- He's vague about when company will be registered
- He pushes back hard on ANY advance
- He says "trust me, I'll pay you later"
- No timeline, no commitment, just "let's discuss more"
