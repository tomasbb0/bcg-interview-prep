## 🗣️ EXPRESSÕES ÚTEIS DURANTE O CASO

### Estruturar:

- "Para responder a esta questão, proponho analisar **três dimensões**..."
- "Vou dividir a minha análise em **dois grandes blocos**..."
- "A minha abordagem seria a seguinte..."
- "Faz sentido esta estrutura? Posso começar por...?"

### Pedir informação:

- "Tem dados sobre...?"
- "Sabe-se qual é a evolução de... nos últimos anos?"
- "É possível ter mais detalhe sobre...?"
- "Há alguma informação adicional que me possa dar?"

### Fazer hipóteses:

- "A minha hipótese inicial é que..."
- "O meu palpite / intuição é que o problema está em..."
- "Vou assumir que... Parece-lhe razoável?"
- "Se eu tivesse que apostar, diria que..."

### Sintetizar:

- "Então, resumindo o que vimos até agora..."
- "Portanto, a principal conclusão é que..."
- "Isto confirma a minha hipótese de que..."
- "Ou seja, o driver principal aqui é..."

### Recomendar:

- "A minha recomendação seria..."
- "Eu recomendaria **três ações principais**..."
- "Com base na análise, sugiro que a empresa..."
- "Os próximos passos seriam: primeiro..., segundo..., terceiro..."
- "Há, no entanto, **riscos a considerar**..."

### Quando ficas preso:

- "Deixe-me recuar um passo e reformular..."
- "Interessante. Antes de avançar, quero garantir que estou a compreender bem..."
- "Posso pensar um momento para organizar a minha abordagem?"
- "Vou fazer uma aproximação e depois refinamos."

### Fazer contas:

- "Deixe-me fazer as contas rapidamente..."
- "Se arredondarmos para facilitar..."
- "Portanto, isto dá-nos aproximadamente..."
- "Vou verificar a sanidade deste número. Faz sentido porque..."

## ⚠️ ERROS COMUNS A EVITAR

| ❌ Não digas               | ✅ Diz antes                                   |
| -------------------------- | ---------------------------------------------- |
| "Isso é um bom framework"  | "Proponho analisar isto da seguinte forma"     |
| "Vou usar profitability"   | "Vou analisar receitas e custos separadamente" |
| "Revenue" (sozinho)        | "Receitas" ou "Faturação"                      |
| "Costs" (sozinho)          | "Custos"                                       |
| "O market share"           | "A quota de mercado"                           |
| "Market sizing" (repetido) | "Dimensionamento do mercado"                   |
| "Pricing" (isolado)        | "Estratégia de preço" ou "Definição de preço"  |
| "Não sei"                  | "Vou fazer uma estimativa baseada em..."       |
| "Isso é difícil"           | "Interessante. Deixe-me pensar numa abordagem" |
