## 📌 FÓRMULAS ESSENCIAIS

### 1. Ponto de Equilíbrio (Breakeven Point)

```
Ponto de Equilíbrio (unidades) = Custos Fixos / (Preço por Unidade - Custo Variável por Unidade)

Ponto de Equilíbrio (€) = Custos Fixos / Margem de Contribuição (%)
```

**Margem de Contribuição** = Preço - Custo Variável por Unidade

**Exemplo:** Custos fixos = €100K, Preço = €50, Custo var. = €30
→ Breakeven = 100.000 / (50 - 30) = 100.000 / 20 = **5.000 unidades**

---

### 2. Período de Retorno (Payback Period)

```
Payback Period = Investimento Inicial / Fluxo de Caixa Anual

(Para fluxos de caixa irregulares: somar ano a ano até recuperar o investimento)
```

**Exemplo:** Investimento = €500K, Cash flow anual = €125K
→ Payback = 500.000 / 125.000 = **4 anos**

---

### 3. Retorno sobre Investimento — ROI (Return on Investment)

```
ROI = (Ganho - Investimento) / Investimento × 100%

ROI = Lucro Líquido / Investimento × 100%
```

**Exemplo:** Investi €200K, ganhei €280K → ROI = (280-200)/200 = **40%**

---

### 4. Margens

```
Margem Bruta = (Receita - COGS) / Receita × 100%
   COGS = Cost of Goods Sold = Custo dos Produtos Vendidos

Margem Operacional (EBIT margin) = EBIT / Receita × 100%
   EBIT = Receita - COGS - Despesas Operacionais
   (Antes de juros e impostos)

Margem Líquida = Lucro Líquido / Receita × 100%
   (Depois de TUDO: juros, impostos, etc.)

Margem de Contribuição = (Preço - Custo Variável) / Preço × 100%
```

---

### 5. Taxa de Crescimento (Growth Rate)

```
Crescimento (%) = (Valor Final - Valor Inicial) / Valor Inicial × 100%

CAGR (Taxa de Crescimento Anual Composta):
CAGR = (Valor Final / Valor Inicial)^(1/n) - 1
   onde n = número de anos
```

**Dica rápida CAGR:** Se algo duplicou em 7 anos → CAGR ≈ 10%. Triplicou em 7 anos → CAGR ≈ 17%

**Regra dos 72:** Para estimar quando algo duplica: 72 / taxa de crescimento = anos para duplicar

- 10% crescimento → duplica em ~7 anos
- 15% crescimento → duplica em ~5 anos
- 7% crescimento → duplica em ~10 anos

---

### 6. Valor Atual Líquido — VAL (NPV - Net Present Value)

```
VAL = Σ [Cash Flow_t / (1 + r)^t] - Investimento Inicial

Simplificação para caso: VAL ≈ (CF anual × n anos × fator de desconto) - Investimento
```

**Conceito que basta saber:** "Um euro hoje vale mais do que um euro amanhã." Se a taxa de desconto é 10%, €100 daqui a 1 ano valem hoje ≈ €91.

**Fator de Desconto:**
| Taxa | 1 ano | 2 anos | 3 anos | 5 anos |
|------|-------|--------|--------|--------|
| 5% | 0.95 | 0.91 | 0.86 | 0.78 |
| 10% | 0.91 | 0.83 | 0.75 | 0.62 |
| 15% | 0.87 | 0.76 | 0.66 | 0.50 |

---

### 7. Quota de Mercado (Market Share)

```
Quota de Mercado = Vendas da Empresa / Vendas Totais do Mercado × 100%
```

---

### 8. Customer Lifetime Value — CLV (Valor do Ciclo de Vida do Cliente)

```
CLV = Receita Média por Cliente × Margem × Tempo Médio de Retenção (anos)

Ou: CLV = ARPU × Margem / Taxa de Churn
   ARPU = Average Revenue Per User (Receita Média por Utilizador)
   Churn = taxa de cancelamento/perda de clientes
```

---

### 9. Custo de Aquisição de Cliente — CAC

```
CAC = Custo Total de Marketing e Vendas / Número de Novos Clientes

Rácio saudável: CLV/CAC > 3 (o valor do cliente deve ser 3x o custo de o adquirir)
```

---

### 10. Elasticidade-Preço

```
Elasticidade = % Variação na Quantidade / % Variação no Preço

Se |E| > 1: Elástico (sensível ao preço — reduzir preço aumenta receita)
Se |E| < 1: Inelástico (pouco sensível — subir preço aumenta receita)
Se |E| = 1: Unitário
```

## 🧮 ATALHOS DE MATEMÁTICA MENTAL

### Percentagens Rápidas

```
10% de X = X / 10
5% de X = X / 20
1% de X = X / 100
15% de X = 10% + 5% = X/10 + X/20
20% de X = X / 5
25% de X = X / 4
33% de X = X / 3
50% de X = X / 2
75% de X = X - 25% de X
```

### Multiplicação Rápida

```
× 5 = × 10 / 2
× 15 = × 10 + × 5
× 25 = × 100 / 4
× 50 = × 100 / 2
× 1.1 = × 1 + 10%
```

### Divisão Rápida

```
÷ 3 ≈ × 0.33
÷ 6 ≈ × 0.167
÷ 7 ≈ × 0.143
÷ 8 ≈ × 0.125
÷ 9 ≈ × 0.111
```


## 📊 RÁCIOS FINANCEIROS ÚTEIS

| Rácio (PT)               | Fórmula                         | Benchmark                                           |
| ------------------------ | ------------------------------- | --------------------------------------------------- |
| Margem Bruta             | (Receita-COGS)/Receita          | Varia: Retalho 30-40%, Tech 60-80%, Serviços 40-60% |
| Margem EBITDA            | EBITDA/Receita                  | 15-25% (bom)                                        |
| Margem Líquida           | Lucro Líquido/Receita           | 10-20% (bom)                                        |
| Rácio Dívida/Capital     | Dívida/Capital Total            | < 40% (saudável)                                    |
| Current Ratio (Liquidez) | Ativo Corrente/Passivo Corrente | > 1.5 (saudável)                                    |
| Rotação de Stock         | COGS/Stock Médio                | Mais alto = melhor                                  |

## 🔑 TERMOS FINANCEIROS PT vs EN

| Português                  | English                       | Explicação                             |
| -------------------------- | ----------------------------- | -------------------------------------- |
| Receitas / Faturação       | Revenue                       | Dinheiro que entra                     |
| Custos / Despesas          | Costs / Expenses              | Dinheiro que sai                       |
| Lucro Bruto                | Gross Profit                  | Receita - COGS                         |
| Resultado Operacional      | Operating Income / EBIT       | Receita - todos custos operacionais    |
| Lucro Líquido              | Net Income / Net Profit       | "Bottom line" — depois de tudo         |
| Fluxo de Caixa             | Cash Flow                     | Dinheiro real que entra e sai          |
| Balanço                    | Balance Sheet                 | Ativos = Passivos + Capital Próprio    |
| Demonstração de Resultados | Income Statement / P&L        | Receitas - Custos = Lucro              |
| Capital Próprio            | Equity / Shareholders' Equity | O que os acionistas possuem            |
| Ativo                      | Assets                        | O que a empresa tem                    |
| Passivo                    | Liabilities                   | O que a empresa deve                   |
| Amortização / Depreciação  | Depreciation / Amortization   | Perda de valor ao longo do tempo       |
| Taxa de Desconto           | Discount Rate                 | Para calcular valor presente           |
| Custo de Oportunidade      | Opportunity Cost              | O que perdes ao escolher A em vez de B |
